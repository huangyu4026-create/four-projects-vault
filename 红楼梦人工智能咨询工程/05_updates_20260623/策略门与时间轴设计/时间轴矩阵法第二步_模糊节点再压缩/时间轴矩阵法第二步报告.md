## 二、二轮口径

第一轮是全库粗矩阵。第二轮只回到内部原文段：

```text
外部节点 -> 事件词/人物词/时间词 -> segments.summary / quote
-> old_segment_no -> atom_coordinates.time_block_id
```

二轮只解决一个问题：

```text
这个外部节点在内部原文里有没有一个比较像的落点？
```

它仍然不能让外部节点进入主轴。

## 三、使用建议

1. `二轮可候选` 可以进入人工/机器原文复验队列。
2. 复验通过后，由内部原文段、时间块、聚点生成主轴点。
3. `二轮压缩后仍模糊` 保留，后续用章节范围、人物年龄、事件链再排。
4. `仍未设定` 暂不使用。
