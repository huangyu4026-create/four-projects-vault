# 数字生命｜GitHub 入口

数字生命工程先作为四项目工程的第一个 GitHub 试运行项目。

## 当前定位

数字生命不是普通写作项目，也不是营销文案项目，而是日常生命档案、事实证据、时空轴、人物关系、日志入库和生命理解的综合工程。

## 本地权威入口

本地权威入口仍然是：

```text
/Users/yu/Documents/Codex/数字生命工程入口.md
```

以及数字生命项目大门：

```text
/Users/yu/Documents/Codex/2026-06-03/notion-3-crv/四项目工程总入口/01_数字生命_项目大门/数字生命_项目大门.md
```

GitHub 第一阶段只作为版本正库试运行，不替代本地权威入口。

## 默认工作方式

用户说“进入数字生命”“数字生命工作”“数字生命接上”时，默认按数字生命状态工作。

若用户已经在数字生命专用对话框中工作，不必每次重复说“进入数字生命”。

## 第一阶段 GitHub 目标

1. 固化目录结构。
2. 固化触发词和状态加载规则。
3. 记录同步水位。
4. 保存处理报告。
5. 暂不上传全量私人日记正文。
