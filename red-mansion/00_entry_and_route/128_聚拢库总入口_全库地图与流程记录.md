## 一句话定案

```text
聚拢库是总入口，也是总归口。
进入工程先过全库地图，不读全库正文。
相关库深盘以后，所有线索必须回到聚拢库。
聚拢库再带编号回原文裁判。
命中只算召回，不算放行；没有聚拢裁判，不得写最终答案。
```

## 一、进入红楼梦工程的真实起手式

外面只需要短短一句，不再长篇宣誓：

```text
已接入红楼梦工程入口。
本题闭环包：package_dir=____。
聚拢四件套：00AC/00AG/00AI/00AM 已齐 / 缺____。
已过库地图：先回聚拢库，相关库深盘，最后原文裁判。
```

如果有具体问题，下一步不是先造词，而是先做流程记录。

如果已经命中材料，下一步仍然不是写答案，而是改写状态：

```text
召回已命中 -> 待聚拢裁判
```

任何“可直接命中”都只能进入召回状态，不能进入最终写作状态。

四件套齐全只说明已经进聚拢库。最终红楼解语必须另验：

```text
00I_Codex材料池判定_<request_id>.md/json
00L_Codex精读材料词_<request_id>.md/json
00M_Codex写作前原文通读摘抄_<request_id>.md/json
```

`00ZI/00ZP` 是 00L/00M 写回后的同步稿位或人读看板，不是最终写作放行证。最终放行只看正式 `00I/00L/00M` 的 md+json 是否齐全。

缺任一项，最终写作状态一律为：

```text
硬阻断：召回已命中，待 00I/00L/00M 闭环。
```


## 一点五、本题闭环包与聚拢四件套

“过全库地图”只是知道库群位置，不等于已经进入聚拢库。

每个具体内容问题必须先有本题闭环包凭证：

```text
package_dir：____
00AC_聚拢库总入口流程锁：有 / 无 / 已读 / 未读
00AG_聚拢库取材单：有 / 无 / 已读 / 未读
00AI_聚拢入材料池清单：有 / 无 / 已读 / 未读
00AM_聚拢裁判首读材料池：有 / 无 / 已读 / 未读
```

这些凭证缺任一项，`formal_honglou_search.sqlite`、segments、search_documents 的命中都只能算底层召回，不能算进入聚拢库。

## 二、流程记录必须有四格

```text
本题闭环包：package_dir=____
聚拢四件套：00AC/00AG/00AI/00AM 已齐 / 缺____
问题类型：人物 / 关系 / 物象 / 空间 / 事件 / 主题 / 文本功能 / 全文穷尽
先过地图：已查看库登记处，知道可用库群
相关库：本题实际要深盘哪些库
回聚拢库：这些库线索映射回哪些编号、聚拢段、聚拢单元、聚拢事件、聚拢场或聚拢域
原文裁判：最终落到哪些回目和原文
召回状态：未命中 / 已命中 / 命中充分 / 命中不足
聚拢裁判状态：未裁判 / 裁判中 / 已裁判 / 裁判不足需补证
最终写作状态：不可写 / 可写草案 / 可写最终答案
```

这不是限制，是防止空口造词。

## 三、总流程

```text
进入红楼梦工程
-> 读 000 根门
-> 创建或确认本题闭环包
-> 自检 00AC/00AG/00AI/00AM
-> 过全库地图
-> 进入聚拢库 / 聚拢总图
-> 判断问题类型
-> 选择相关库深盘
-> 从相关库取得规范名、标签、入口词、候选编号
-> 全部映射回聚拢库
-> 在聚拢库内放大、缩小、交集、串域
-> 回原文裁判
-> 入材料池
-> 材料池四态判定
-> 精读材料词 / 聚拢裁判单
-> 写作前原文追证摘抄
-> 聚拢裁判通过后红楼解语 / 终稿复核
```

状态机硬规则：

```text
召回状态不能直接升级为最终写作状态。
聚拢裁判状态不是“已裁判”，最终写作状态只能是“不可写”或“可写草案”。
材料池四态未完成，不能写最终红楼解语。
00I/00L/00M 三件终答前置未完成，不能写最终红楼解语。
```

## 五、聚拢库的总归口地位

所有库都不是终点。

```text
人物库 -> 回聚拢库
关系库 -> 回聚拢库
物象库 -> 回聚拢库
空间库 -> 回聚拢库
事件库 -> 回聚拢库
主题库 -> 回聚拢库
全文穷尽工具 -> 回聚拢库
```

聚拢库负责把线索变成可判断的现场：

```text
编号
聚拢段
聚拢单元
聚拢事件
聚拢场
聚拢域
```

最后再回原文。

## 六、不同问题的相关库深盘方向

| 问题类型 | 先深盘的相关库 | 回聚拢库后做什么 |
|---|---|---|
| 人物题 | 人物基础库、人物-段落映射库 | 归一人物，收人物编号点，再回聚拢场和原文 |
| 人物关系题 | 人物关系主库、人物-段落映射库、跨轴证据边库 | 找关系证据点，回聚拢事件/场判断关系强弱 |
| 同场题 | 人物-段落映射库、聚拢单元库、聚拢事件库、空间证据库 | 求交集，放大到聚拢场，再判真实同场/同回不同场 |
| 物象题 | 物象器物库、物件流转库、全文穷尽查证工具库 | 物象标签收点，回聚拢事件/场看流转 |
| 空间题 | 空间场所与空间证据库、人物空间出入库、回目时间场库 | 空间归一，回聚拢场判断谁在何处做什么 |
| 事件题 | 事件片段与事件段落库、聚拢事件库 | 事件编号回聚拢单元/场，再回原文 |
| 主题题 | 对应专题库、文学文本轴库、全文穷尽查证工具库 | 主题标签先回聚拢域，再落回具体场和原文 |
| 文本功能题 | 文学文本轴库、原文章节库、全文穷尽查证工具库 | 诗词/判词/书信/梦境等先定位，再回原文 |
| 全书穷尽题 | 全文穷尽查证工具库、原文章节库、段落库 | 先全书收点，再回聚拢库过滤，不直接出结论 |
