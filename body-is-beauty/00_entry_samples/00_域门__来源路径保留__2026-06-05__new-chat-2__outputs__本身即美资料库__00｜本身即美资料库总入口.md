# 00｜本身即美资料库总入口

## 定位

本资料库用于承接“本身即美 / 小红书 / 愤怒潮湿”的全部研究材料。

它不是把每篇文章拆成细碎数据库，而是把每篇文章、每份讨论、每个文案、每个判断作为一个资料对象入库，再用统一指标进行检索、阶段归类、类型归类和后续分析。

## 分库结构

| 分库 | 作用 |
|---|---|
| 01｜文案库 | 保存成品文案、候选文案、文案风格、标题、发布版正文 |
| 02｜思考库 | 保存用户口述、AI讨论、过程判断、阶段想法、协作偏好 |
| 03｜论证库 | 保存为什么这样分阶段、为什么这样判断、爆点/限流/转向依据 |
| 04｜定义库 | 保存概念定义、阶段定义、类型定义、标签定义、字段定义 |
| 05｜笔记对象库 | 保存每条小红书笔记作为对象的索引 |
| 06｜数据反馈库 | 保存点赞、时间、主页读取、爆点数据、限流节点 |
| 07｜产品库 | 保存愤怒潮湿、短裤、泳裤、产品推荐规则 |
| 08｜阶段库 | 保存 P00-P14 阶段、阶段摘要、阶段演化 |
| 09｜来源库 | 保存原始材料、主轴、暂存区、粘贴文本、读取回执 |
| 10｜查询协议库 | 保存查询路径、纠错机制、入库规则 |

## 使用路径

以后查询时，先按这个顺序：

1. 查 `05｜笔记对象库`，确定对象。
2. 查 `08｜阶段库`，确定阶段。
3. 查 `01｜文案库` 或 `02｜思考库`，看正文和讨论。
4. 查 `09｜来源库`，回到原始材料。
5. 查 `03｜论证库`，确认判断依据。
6. 查 `04｜定义库`，确认标签和概念口径。
