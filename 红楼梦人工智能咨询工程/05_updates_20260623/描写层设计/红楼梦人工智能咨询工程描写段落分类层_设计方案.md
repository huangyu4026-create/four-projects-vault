# 红楼梦人工智能咨询工程描写段落分类层设计方案

生成时间：2026-06-23

## 粗查结论

只读坐标查询唯一中心大库检查后，现有大库已经有部分描写相关线索，但还没有形成完整的“描写段落分类库”。

已存在的相关表头和数量：

```text
function=环境描写          143
function=服饰描写           10
function=心理独白          223
perspective=人物内心       121
emotion_symbolic_function=外貌 40
emotion_symbolic_function=服饰 14
note_type=外貌              17
note_type=服饰               8
note_dimension=环境        116
```

判断：这些可以作为现成线索，但不够回答“全书哪些段落是描写衣服、容貌、景物、动作、心理”等系统问题。需要补一个正式的描写段落分类层，并回投坐标查询唯一中心大库。

## 目标

把全书每个原子段过一遍，判断它是否含有描写；如果含有，标明描写类型、描写对象、触发词、原文摘录和坐标。

查询时仍只进入坐标查询唯一中心大库，用新的 `description_*` 表头/编码标取材。

## 建议分类

```text
description_appearance        容貌描写：脸、眉、眼、身量、风姿、神态、体态。
description_clothing          衣服服饰描写：衣、裙、袄、褂、冠、钗、环、佩饰、穿戴。
description_scenery           景物景色描写：花、树、水、月、雪、风、园景、山石。
description_space_environment 空间环境描写：房间、院落、陈设、门窗、床榻、器具布置。
description_psychology        心理描写：心想、暗想、心下、疑心、忖度、悒郁、惊疑。
description_action            动作描写：走、坐、看、笑、哭、拉、递、拜、跪、掀、藏等身体动作。
description_expression        神情声态描写：笑容、泪、怒色、低声、叹、叫、哭、忙、慌。
description_object_detail     器物细描：玉、扇、帕、香囊、药、杯盘、镜、帐、枕等被细写。
description_sense_color       颜色/感官描写：红、白、香、冷、暖、明、暗、声、气味。
description_mixed             多类混合描写：同一原子段里多种描写并重。
```

## 入库模型

新层不另立查询入口，最终写回坐标查询唯一中心大库。

建议新增三张同库表：

```text
description_passage_cards
```

## 实现方法

第一步：只读坐标查询唯一中心大库的 `atom_coordinates`，按 `global_atom_order` 从第 1 个原子段到最后一个原子段逐段处理。

第二步：用 `atom_coordinates.quote` 和 `summary` 做初判；必要时用坐标回跳原文核准。

第三步：每个原子段都生成一张 `description_passage_cards`。没有描写也要记录 `has_description=0`，保证全量过段。

第四步：有描写的原子段，按触发词和对象生成 `description_axis_points`。

第五步：把新层写回坐标查询唯一中心大库，并在 `build_meta` 写入：

```text
description_layer_integrated_at
description_layer_source
description_layer_policy
```

第六步：做粗验收：

```text
description_passage_cards 行数 = atom_coordinates 行数
description_axis_points > 0
description_* variable_points > 0
integrity_check = ok
```

## 查询能力

完成后可直接回答：

```text
全书哪些段落是容貌描写？
王熙凤初次出场有哪些服饰描写？
林黛玉相关的心理描写集中在哪些回？
大观园有哪些景物描写？
某一回有哪些动作描写？
某个物件被细写在哪些段落？
某场景里同时有容貌、服饰、环境描写的原子段有哪些？
```

## 注意

现有大库里的 `function=环境描写/服饰描写/心理独白` 等只能作为线索，不等于完整描写层。正式描写层必须全量逐原子段处理，并投射回坐标查询唯一中心大库。
