## 工程定位

- 红楼梦工程：保留旧聚拢/闭环入口和完整后半身。
- 坐标工程：独立呈现坐标入口、坐标总库、变量投影、距离/共场/交集、候选入材料池的前半身。
- 两者关系：坐标工程是红楼梦工程的另一个查询版本；坐标候选后半身仍要回到材料池、原文裁判、精读材料词、写作前原文追证和红楼解语。

## GitHub 内保存什么

GitHub 保存入口、规则、脚本、验收、schema、表行数、样例和大文件清单。这样新线程或新机器能知道工程怎么进、底层库是什么、哪些东西必须回本地找。
