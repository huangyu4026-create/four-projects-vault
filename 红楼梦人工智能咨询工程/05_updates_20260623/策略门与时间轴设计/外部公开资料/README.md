# 外部公开资料

本文件夹用于存放红楼梦人工智能咨询工程可公开引用的外部资料。

资料进入本文件夹的条件：

| 条件 | 说明 |
|---|---|
| 公开来源 | 来自公开网页、公开论文、公开数据库或可验证出版资料 |
| 高价值 | 能帮助校准时间轴、人物年表、空间路线、事件顺序、版本争议或原文查证 |
| 可回链 | 必须保留来源链接、下载时间或原始文件 |
| 不进主轴 | 只做外部旁证队列和复核提示，不能替代原文证据，不能写入主时间轴 |
| 可复核 | 关键结论要能被回查、比对、标注争议 |

当前资料：

| 文件 | 来源 | 用途 |
|---|---|---|
| `红楼梦年谱_wikipedia.html` | https://zh.wikipedia.org/wiki/红楼梦年谱 | 外部年谱参照；提供“前八十回/后四十回”和相对年份模型 |
| `红楼梦年谱_wikipedia_抽取文本.txt` | 同上网页抽取文本 | 便于工程读取、比对、标注时间节点 |
| `红楼梦年谱_wikipedia_外部年份模型.csv` | 同上网页结构化 | 外部年份模型，当前 201 条正文节点 |
| `红楼梦年谱_wikipedia_与现有time_axis候选对齐.csv` | 外部年份模型 + 现有 `time_axis` 粗对齐 | 候选对齐表，当前 281 条；只作召回，必须回原文复核 |

相关矩阵法输出：

| 目录 | 用途 |
|---|---|
| `../时间轴矩阵法第一步/` | 外部节点对内部 `time_axis / segments / time_blocks / variable_points` 做第一轮矩阵 |
| `../时间轴矩阵法第二步_模糊节点再压缩/` | 对第一轮模糊/未设节点回到原文段做二轮压缩 |
| `../时间轴矩阵法第三步_夹逼复核/` | 对高价值模糊节点用前后内部节点夹逼 |
| `../时间轴外部旁证矩阵法_三轮总账/` | 三轮总复验队列，当前 49 条进入复验队列，152 条旁路保留 |

使用原则：

```text
原文时间层最高。
工程推论层其次。
外部公开资料只做旁证、补漏和争议提示；外部时间节点不能进入主轴。
```
