这是“四项目工程”的 GitHub 试运行仓库。当前只先接入“数字生命”一个项目。GitHub 用于保存规则、状态加载、触发词、同步水位和处理报告；Notion 继续作为公共收件箱和人读入口；本地总账仍是判断是否真正处理完成的最终依据。
