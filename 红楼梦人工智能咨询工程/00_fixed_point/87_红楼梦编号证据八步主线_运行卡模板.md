# 87_红楼梦编号证据八步主线_运行方法卡

生成时间：2026-06-20

用途：每个红楼梦问题进入工程时，先读本方法卡。八步法是完整方法，覆盖读题、取词、归一、收点、交集、查库、回原文和入材料池。

硬定位：本文件是“八步方法”，不是写作模板，也不是改名入口。它规定 Codex 如何把用户问题拆成可查对象，如何利用库表和编号工作，如何把候选送入材料池。

## 0. 八步法基础上的两个思考区

```text
第一部分：取词之前的思考。
目的：读懂题目，拿到主查词、归一词、强复合轴、扩展词、背景词、排除词，并写清查询词策略。
位置：八步法内部前段，在入口词/取词阶段完成。

第二部分：取词以后如何用库的思考。
目的：带着已经取好的词，决定查哪个库、怎样收点、怎样交集、怎样回原文、怎样入材料池。
位置：八步法内部后段，在查库、交集、回原文、入材料池阶段完成。

这两个部分都建立在完整八步法之上，不是另起两个流程，也不是把八步法拆散。
```

## 0A. 取词之前的思考

```text
用户问题 -> 判断题型 -> 拆对象 -> 定主轴 -> 定查询词策略 -> 取词 -> 形成入口词包
```

必须完成：

```text
1. 判断题目是不是人物、物象、空间、季节、事件、关系、距离、共场、第一次、全文穷尽。
2. 找出题目真正要查的对象，不把“进入工程”“坐标查询”“帮我查”等路由词当内容词。
3. 人名先走人物归一意识：用户说的人名、别名、称谓必须归到人物包。
4. 物象词先判断是具体物、物象族、诗词意象、场景提示，不能只按单字粗暴扩散。
5. 多对象题要形成强复合轴：谁和谁、人与物、人与季节、人与地点、人与事件，哪些必须共同成立。
6. 写出查询词策略：中心词、强复合、扩展词、背景锚点、排除词为什么这样取舍。
7. 生成入口词包后，在同一套八步法里进入“取词以后如何用库”的思考。
```

## 0B. 取词以后如何用库的思考

```text
不管走语义聚拢还是坐标查询，都必须按完整八步法工作。

入口词包形成后，才允许：
1. 判断本题需要人物、物象、空间、时间、事件、词位、距离、共场中的哪些库轴。
2. 进入对应中心库取材。
3. 把命中材料送入材料池。

禁止：
1. 跳过八步法，直接查库。
2. 只凭命中词写答案。
3. 把同回共现当同场。
4. 把摘要、标签、候选当原文裁判。
```

## 2. S1 归一

```text
人物：
物象：
空间：
事件：
关键词 / 抽象概念：
歧义：
```

方法说明：
- 人物先走人物归一，名字、别名、称谓不得靠猜。
- 物象先判断是硬物象、物象族、场景提示还是裸词；硬物象优先查对象轴，未登记物象再走词位穷尽补点。
- 季节、空间、时间、事件不能和同字人名混用；例如“春天”和“惜春”必须分轴。
- 抽象概念先转成可查对象、动作、情绪词或场景条件，再回原文裁判。

方法说明：
- 收点不是收答案，是把对象落到 segment_no、atom_code、cluster_unit、event_id、scene_id、time_block_id、chapter_no。
- 语义聚拢路优先回语义聚拢中心库的段、单元、事件、人物、物象、空间、证据边。
- 坐标查询路优先回坐标中心库的 atom_coordinates、variable_points、container_variable_index、距离视图和上下文视图。
- 词位穷尽只负责找正文落点；找到后必须带 atom/segment 回中心库和原文。

## 4. S3 交集

```text
segment_no 交集：
cluster_unit 交集：
event_id 交集：
chapter_no 共现：
交集等级：
```

方法说明：
- 交集是找候选门，不是定结论。
- 优先级：同原子段 / 同 segment 最强；同场、同聚拢单元次之；同事件再次；同回只能作背景。
- 多对象题先做两两关系，再看三轴或多轴是否能共同成立。
- 最近、靠近、相邻题必须记录距离层级和距离数值，不能只写“同一回”。

## 5. S4 路由门

```text
关键编号：

现场空间：
现场物象：
现场事件：
相邻段落：
可继续追索路径：
```

方法说明：
- 路由门负责打开现场：谁在、哪里、什么物、什么动作、前后场是什么。
- 候选进入现场后，要区分真实在场、被提及、回忆、梦境、转述、远端影响、象征和反证。
- 聚拢路打开聚拢场；坐标路打开 atom/scene/event/time_block/container；两边最后都要回原文。

方法说明：

- 人物关系题重点看对话、动作、同场、送物、看见、听见、转述。
- 人和物题重点看物是否真实出现、是否被持有/赠送/观看/提及/象征。
- 时间季节题要区分章回背景、场内时间、诗词意象和人物名中的同字。

- 原文裁判至少要看命中句和必要上下文；摘要、标签、变量名不能直接作结论。
- 若原文只支持背景或象征，必须降级，不准升主证。
