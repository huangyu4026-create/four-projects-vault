## 一句话定案

以后“进入红楼梦工程”只认这一条运行顺序：

```text
先读根门。
先建本题闭环包。
先验聚拢四件套。
先守对象。
先判类型。
先出入口词。
再入聚拢总图。
图内盘路。
回原文裁判。
命中只算召回。
聚拢裁判后才进入最终写作。
```

任何旧文档里出现“先进图再判题”的表述，都按本文件收束。

## 唯一运行顺序

```text
0. 000 根门：确认接入红楼梦工程入口，禁止自由搜索。
0A. 本题闭环包：运行或确认 formal_honglou_closed_loop.py，取得 package_dir。
0B. 聚拢四件套自检：00AC/00AG/00AI/00AM 必须存在并已读。
1. 128 全库地图：知道库群位置，但不读全库正文。
2. 126 查询词规范：人物先归一；物象、小词、动作、文本功能才查根字。
3. 130 第一层：外部判题打卡，判题型、对象、边界。
4. 生成入口词：按 126 通用规则生成题面词、归一人物、物象/文本功能根字、绑定词。
5. 聚拢总图 runtime：加载 113 总图和 118 映射层。
6. 130 第二层：入口词与入图盘面打卡。
7. 119 图内读法：节点身份、标签身份、放大/缩小/交集/串域。
8. 现成编号入口门 / 穷尽补点：只补编号，不直接出答案。
9. 118 映射分流：强证/直连候选、候选方向、需复审问题队列。
10. 原文裁判：落回回目、segment_no、原文现场。
11. 入材料池：候选只入池，不直接写答案。
12. 材料池四态判定：可用主证 / 背景材料 / 不可用 / 需补证。
13. 聚拢裁判单 / 精读材料词：把候选关系、排除项、证据等级裁开。
14. 写作前原文追证摘抄：把最终写作支点回到原文。
15. 红楼解语 / 终稿复核。
```

状态机硬锁：

```text
没有本题 package_dir，不进入查询。
缺 00AC/00AG/00AI/00AM，不算进入聚拢库。
召回状态不得直接升级为最终写作状态。
没有聚拢裁判单，不写最终红楼解语。
材料池四态未完成，不写最终红楼解语。
写作前原文追证摘抄未完成，不写最终红楼解语。
```

## 双打卡判定标准

### 红灯退回

```text
无题型。
无对象。
无入口词。
```

红灯时不准改走自由搜索。

### 黄灯可继续但必须补卡

```text
入口词含人名字根。
物象 / 文本功能题没有经过通用词根拆解，或拆出的根字没有进入入口词。
118 映射候选过大。
118 需复审比例高。
118 未输出强证/直连、候选、需复审三档数字。
tag_hits = 0 但 node_hits / line_hits 有命中，需要说明标签层未命中。
候选为 0，需要全文穷尽补编号。
显式深度题或多论证题未拆题。
```

黄灯可以继续，但必须写明补卡动作。

### 绿灯通过

```text
题型清楚。
对象清楚。
入口词清楚。
对象类型、词根拆解路径、绑定条件已进入入口词记录。
图内盘法清楚。
有原文出口。
没有高噪音，或已说明分流办法。
```

## 查询词硬规则

```text
人物先归一，不查人名字根。
人物关系题先归一双方，再做交集 / 关系弧。
物象、小词、动作、文本功能按通用规则拆最小根字。
物象根字只负责收候选，不准直接出答案，也不准为单题硬塞词。
所有候选编号必须回聚拢总图。
所有强结论必须回原文裁判。
118 强证/直连候选也只是入口，必须回原文裁判。
118 候选方向不得直接写成答案。
118 需复审只能进问题队列或补审路径。
任何“可直接命中”都只能写成“召回已命中，待聚拢裁判”。
```

## 三题验收基线

### A. 人物关系 / 同场题

题目：

```text
贾雨村和林黛玉面对面交流有哪些？贾雨村和贾政碰面交流有哪些？关系走势怎样？
```

必须达到：

```text
问题类型：同场 / 人物关系
人物：林黛玉、黛玉、贾政、贾雨村、雨村
物象：无
不允许出现人名字根“玉”
应触发子问题预览：是
触发原因：题内有三个论证单元：贾雨村-林黛玉、贾雨村-贾政、关系状态判断
状态：黄灯可继续
黄灯原因：118 映射召回大，需分强证 / 候选 / 复审
```

### B. 文本物象 / 书信题

题目：

```text
林如海给贾雨村写的荐书这封信，在后面120回里有没有流转或再出现？起什么作用？
```

必须达到：

```text
问题类型：物象 / 文本功能 / 人物
人物：林如海、贾雨村、雨村
物象/文本功能对象：林如海荐书 / 书信
通用词根拆解：文本功能题应自然产出“书、信”等根字
入口词记录必须说明：题面对象、对象类型、根字来源、绑定人物
如果没有经过通用词根拆解而直接拿长句入图：黄灯，不可绿灯
不得为本题硬编码“荐书 / 书 / 信”，样题只用于验收通用规则是否生效
```

### C. 物象包住人物名题

题目：

```text
通灵宝玉在全书如何流转？
```

必须达到：

```text
问题类型：物象 / 全文穷尽
人物：无
物象：通灵宝玉、玉
不误拉人物：宝玉、贾宝玉
状态：黄灯可继续或绿灯，视候选噪音而定
```

## 失败即退回规则

```text
人物题拆出人名字根，退回 126。
物象 / 文本功能题没有走通用词根拆解，退回 126。
没有本题 package_dir，退回 000/闭环包入口。
缺 00AC/00AG/00AI/00AM，退回聚拢四件套入口。
没有外部判题，退回 130 第一层。
没有入口词盘面，退回 130 第二层。
没有回聚拢总图，退回 119。
没有原文裁判，不准出强结论。
没有材料池四态，不准写红楼解语。
没有聚拢裁判单，不准写红楼解语。
```
