# 《此生何为》桌面课堂工作台

这个页面参照了“哲思之旅”桌面课堂的结构，但做成轻量版：

- 上方是主讲区，显示当前讲次、开场、三问/五问和课程推进路径。
- 顶部有“上课 / 下课”两个正式触发按钮：上课进入当前讲次，显示开场；下课生成收束整理请求。
- 顶部和状态卡都可触发“上课 / 下课”；“开始第5讲”会直接进入第5讲上课流程。
- “检查回应”会按最近一次 `request_id` 读取结果，并把 Codex 写回的答案显示到主讲区。
- “处理队列提示”会显示 Codex 侧触发词，避免问题提交后不知道下一步怎么走。
- “复制触发词”会复制 `处理此生何为课堂队列`；如果浏览器不允许复制，页面会把触发词显示出来供手动复制。
- 如果浏览器没有保存最近编号，“检查回应”会从轻量回显索引中读取最近一条结果。
- 下方是回答区。通过 `启动此生何为课堂.command` 打开后，可以把问题写入本地召回队列。
- 资料卡按底稿、三问、五问、流程、状态、课堂记录展开。
- 轻量召回区能搜索章节、资料卡和课堂记录，保证每个按钮都有文字回显。
- 本地服务会生成 `request_id`，写入课堂收件箱、召回队列和轻量回显状态。

```text
  -> 页面“检查回应”
  -> 主讲区显示正式课堂回应
```
