## 四、口径

```text

```

## 六、下一步

优先处理 `内部强证可设点` 和 `内部证据可候选`：

1. 逐条打开 `best_old_segment_no / best_atom_code / best_time_block_id`。
2. 回原文窗口确认时间、事件、人物是否同一处。
3. 确认后，由内部证据生成主轴时间点。
4. 外部节点只写入复核表的 `external_node_id`，不写入主轴。
5. `模糊待复核` 和 `暂未设定` 保留旁路，后续用人工、章节范围、人物年龄、事件链再处理。
